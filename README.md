### Install PostgreSQL database adapter for Python
```bash
pip install psycopg

# Использование python -m гарантирует, что пакет попадет ровно в тот venv, который сейчас запущен в вашем терминале
python -m pip install --force-reinstall "psycopg[binary]"
```

### Install third-party HTTP library:
```bash
pip install requests
```

### Install fake-useragent (web browser User-Agent)
```bash
pip install fake-useragent
```

### Install OpenAI
```bash
pip install openai
```
